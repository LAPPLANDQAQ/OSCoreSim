# OSCoreSim 发行版运行说明

## 运行环境

- Windows 10 / 11 64 位
- 无需安装任何运行时（程序为静态链接 MSVC Runtime）

## 目录结构

```text
发行版/
├── os_sim.exe              # 可执行程序
├── data/
│   └── os_state.bin        # 二进制快照文件（首次运行自动生成）
├── tests/
│   ├── full_demo_commands.txt
│   └── ...                 # 验收测试脚本
└── 运行说明.md
```

## 启动方式

```powershell
.\os_sim.exe
```

## 关键说明

### 1. 状态文件

程序使用相对路径 `data/os_state.bin` 保存和加载快照。

**`data/` 目录必须与 `os_sim.exe` 位于同一级目录下。**

### 2. 首次运行

如果 `data/os_state.bin` 不存在，程序会显示：

```text
[BOOT] No state file found. Starting with a clean system.
```

这是正常现象。

### 3. 生成初始状态文件

```text
register demo 123456
login demo 123456
create_pcb init 64 0 20
save
exit
```

### 4. 脚本演示

```cmd
cmd /c ".\os_sim.exe < tests\full_demo_commands.txt"
```

### 5. 双窗口演示

1. 打开第一个终端，运行 `.\os_sim.exe` → 显示 "角色: MASTER"
2. 打开第二个终端，运行 `.\os_sim.exe` → 显示 "角色: CLIENT"
3. Client 窗口输入命令 → 自动转发到 Master 执行
4. Client 输入 `exit` → 只关闭 Client，Master 继续运行
5. Master 输入 `save` → `exit` → 正常保存退出

## 中文菜单

交互模式下程序会显示中文数字菜单：

- 1-8：功能分类菜单入口
- 9：进入原始命令模式（命令行输入）
- 0：退出程序

脚本重定向输入时自动跳过菜单，直接进入命令模式。
