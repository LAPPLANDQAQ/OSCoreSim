# 报告截图清单

建议在以下场景截取运行截图，插入课程设计报告。

## 启动与基础

| 序号 | 场景 | 操作 |
|------|------|------|
| 1 | 程序启动中文菜单 | 直接运行 `os_sim.exe`，截图启动画面和数字菜单 |
| 2 | help 命令 | 菜单选 9 进入命令模式，输入 `help`，截图完整命令列表 |
| 3 | status 状态 | 输入 `status`，截图内核状态输出 |
